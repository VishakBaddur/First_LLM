# Interactive Code Tools

Interactive Code Tools is a Python-based application designed to assist developers with generating and analyzing code using a local instance of the Mistral-7B Instruct model. The tool supports a range of programming languages, including Python, JavaScript, Java, HTML/CSS, React, and Node.js, offering features tailored to streamline development and improve code quality.

Features
Code Generation: Generate complete, functional code snippets based on user-defined prompts. The generated code includes documentation, error handling, and usage examples, adhering to industry best practices.
Code Analysis: Analyze existing code for performance optimizations, security vulnerabilities, stylistic improvements, or bug fixes. Choose from comprehensive analysis or focus on specific areas, such as optimization or security.
Customizable and Extendable: The application allows you to easily extend the set of supported languages and customize analysis types to suit your needs.
Offline Capability: By using a local instance of the Mistral model, the tool ensures privacy and minimizes reliance on external services.
Requirements
To use this tool, you'll need to download the Mistral-7B Instruct model, which the application can handle automatically if the model is not already present in your system.

Simply run the script and follow the interactive prompts to generate or analyze code efficiently!
